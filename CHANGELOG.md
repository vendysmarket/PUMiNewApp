# Changelog

All notable changes to PUMi will be documented in this file.

## [1.0.0] - 2026-02-10

### Added
- Initial release of PUMi app
- GenZ and Millennial personality modes
- Focus Mode with 45-minute structured sessions
- Interactive lessons, quizzes, and practice exercises
- Real-time AI chat with persistent memory
- Google authentication via Supabase
- Stripe subscription integration
- Multi-language support (Hungarian, English)
- Mobile-responsive design
- Dark mode support
- Account deletion functionality
- Daily streak tracking
- Progress persistence

### Features

#### Chat System
- Real-time AI conversations
- Context-aware responses
- Image upload support
- Session management
- Mobile-optimized UI

#### Focus Mode
- Structured learning plans
- Interactive quizzes
- Translation practice
- Roleplay scenarios
- Writing exercises
- Flashcard reviews
- Progress tracking

#### Authentication
- Google Sign-In
- Session management
- Protected routes
- Secure token handling

#### Billing
- Two subscription tiers (GenZ, Millennial)
- Stripe checkout integration
- Customer portal access
- Webhook handling

### Technical Stack
- Frontend: React 18, TypeScript, Vite, Tailwind CSS
- Backend: Python 3.12, FastAPI
- Database: Supabase (PostgreSQL)
- AI: Claude API (Anthropic)
- Auth: Supabase Auth
- Payments: Stripe
